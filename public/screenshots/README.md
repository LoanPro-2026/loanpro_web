# Screenshot placement guide

Place screenshots in these exact folders and names so the website resolves them correctly.

## Hero section image
- Folder: `public/screenshots/hero/`
- File name: `hero-dashboard.png`
- Final URL used by site: `/screenshots/hero/hero-dashboard.png`

## Download page gallery images
- Folder: `public/screenshots/download/`
- `dashboard.png` → Main Dashboard
- `add-new-record.png` → Add New Record
- `view-accounts.png` → Account Management
- `daily-report.png` → Daily Reports
- `stock-details.png` → Investment Tracking
- `dashboard-dark-mode.png` → Dark Mode

Final URLs used by site:
- `/screenshots/download/dashboard.png`
- `/screenshots/download/add-new-record.png`
- `/screenshots/download/view-accounts.png`
- `/screenshots/download/daily-report.png`
- `/screenshots/download/stock-details.png`
- `/screenshots/download/dashboard-dark-mode.png`

## Notes
- Use lowercase names with hyphens exactly as above.
- PNG is preferred for UI screenshots.
- Recommended hero image ratio: 16:10 or 16:9.
- Recommended download gallery image ratio: 16:9.
